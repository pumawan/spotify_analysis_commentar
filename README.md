# 📌 Overview
This repository contains a Python-based data science project dedicated to analyzing user comments and feedback about the Spotify app. Leveraging Natural Language Processing (NLP) and Machine Learning, the project extracts valuable insights from user reviews to understand user satisfaction (positive or negative).

# 📊 Key Features
1. Data Manipulation & Cleaning: Utilizing Pandas to preprocess text data, handle missing values, and structure raw commentary for analysis.
2. Exploratory Data Analysis (EDA): Beautiful and informative data visualizations crafted with Matplotlib and Seaborn to showcase comment lengths, word frequencies, and rating distributions.
3. Natural Language Processing (NLP): Powered by NLTK for tokenization, stopword removal, lemmatization, and sentiment scoring of user feedback.
4. Machine Learning / Text Classification: Implementing scikit-learn to build predictive models that categorize comments into specific topics or sentiment classes (e.g., bug reports, feature requests, positive praise).

# 🛠️ Tech Stack
> Language: Python
> Data Analysis: Pandas
> Visualization: Matplotlib, Seaborn
> Machine Learning: scikit-learn
> NLP Toolkit: NLTK
